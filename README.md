## Описание

Репозиторий содержит второе домашнее задание по генерационным задачам в NLP. Основная задача — **перенос стиля речи Йоды на русском языке**: преобразование обычных русских предложений в предложения с инвертированным порядком слов, характерным для речи Йоды, с помощью дообученной языковой модели.

---

## Постановка задачи

Речь Йоды отличается нестандартным порядком слов и высоким стилем. Цель задания: обучить модель, которая по обычному русскому предложению генерирует его «йода-версию».

**Пример:**

| Входное предложение | Выход (стиль Йоды) |
|---|---|
| Я пойду в магазин завтра. | В магазин завтра я пойду. |

---

## Модель

В качестве базовой языковой модели используется **Qwen**, дообученная на русскоязычном датасете стиля Йоды. Веса адаптера и конфигурация chat template хранятся в директории `yoda_ru_qwen/`.

## Содержание ноутбука

Основной ноутбук (`HW_NLP_TASK_2_Agapov_KS.ipynb`) включает:

1. Подготовка датасета — формирование параллельного корпуса обычных и «йода-стилизованных» предложений
2. Настройка модели — загрузка Qwen с кастомным chat template (Jinja)
3. Дообучение — supervised fine-tuning на задаче переноса стиля
4. Инференс — генерация выходов в стиле Йоды для тестовых предложений
5. Оценка — качественный и количественный анализ результатов

## Использование

Запустите ячейки ноутбука `HW_NLP_TASK_2_Agapov_KS.ipynb` последовательно. Дообученную модель можно загрузить из директории `yoda_ru_qwen/` для инференса без повторного обучения:

```python
from transformers import AutoModelForCausalLM, AutoTokenizer

model = AutoModelForCausalLM.from_pretrained("./yoda_ru_qwen")
tokenizer = AutoTokenizer.from_pretrained("./yoda_ru_qwen")
```
